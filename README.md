# 🧹 Customer Data Cleaning Project

🚀 A Python project to clean, analyze and generate reports from messy customer data.

## 📌 Description
This project cleans and analyzes customer data using Python and Pandas.

## 🚀 Features
- Removes duplicate data
- Handles missing values
- Separates invalid data
- Generates Excel report
- Shows top cities

## 🛠️ Tools Used
- Python
- Pandas
- Excel

## 📂 Files Included
- main.py (Main script)
- customer_raw_data.csv (Input file)
- clean_customer_data.csv (Clean data)
- invalid_customer_data.csv (Invalid data)
- customer_report.xlsx (Output report)
